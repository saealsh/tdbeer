# تـدّبير 📊
## مخططك المالي الذكي

تطبيق PWA عربي لتتبع المصاريف والأهداف المالية.

🔗 **الموقع:** [saealsh.github.io/tdbeer](https://saealsh.github.io/tdbeer)

---

## ✨ الميزات

- 💰 تتبع الدخل والمصاريف (ثابتة ومتغيرة)
- 🎯 أهداف مالية مع تواريخ نهائية وتقدم مرئي
- 🛡️ ميزانيات بتنبيهات عند 90٪ و100٪
- 💱 دعم 10 عملات خليجية وعالمية
- 🕌 حاسبة زكاة (مالكي وحنفي)
- 🏆 نظام إنجازات و streak يومي
- 👥 Social — تواصل مع الأصدقاء ومشاركة التقدم
- 🤖 AI Bot للنصائح المالية الشخصية
- 📊 Charts تفاعلية (Bar + Donut)
- 📅 مصاريف متكررة تلقائياً كل شهر
- 🔍 بحث في البنود مع تمييز النص
- ⌨️ اختصارات لوحة المفاتيح
- 📴 دعم Offline كامل (PWA)
- 🔒 نسخ احتياطية مشفرة (.tdb)

## 🔒 الأمان

- ✅ XSS protection شامل (DOM nodes بدل innerHTML)
- ✅ Content Security Policy
- ✅ Firestore Security Rules صارمة
- ✅ Rate Limiting على تسجيل الدخول
- ✅ Object.freeze على الثوابت

## 🛠️ التقنيات

- Vanilla JS (no framework)
- Firebase (Auth + Firestore)
- Service Worker (offline + push notifications)
- PWA مع installable manifest

## 📁 الملفات

```
index.html   — التطبيق الكامل
sw.js        — Service Worker
```
